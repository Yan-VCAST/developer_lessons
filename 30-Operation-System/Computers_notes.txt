#2 -- CPU的2大种类
		1.精简指令集RISC： Reduced Instruction Set Computing
			ARM：目前世界上使用范围最广的CPU；手机、导航、交换机路由器等
			IBM的PowerPC：Sony公司的PS3
			Sun公司的SPARC系列：常用于学术领域的大型工作站，包括银行金融体系的主要服务器
		2.复杂指令集CISC
			AMD, Intel， VIA所开发的X86架构CPU被大量使用于个人计算机
			为什么叫X86？最早Intel开发的CPU代号为8086，后来又开发80286、80386等，因此这种架构CPU就被称为X86架构
			不同的X86架构的CPU，除整体结构（二层缓存，每次执行的指令数等），主要在于“微指令集”不同
			最新的Intel/AMD的X86架构，多出了“多媒体微指令集”，“虚拟化微指令集”， “省电功能”， “32/64位兼容技术”

======================

#3 -- 多核：在一个CPU封装当中嵌入了2个以上的运算内核，简单说，就是一个实际的CPU外壳中含有2个以上的CPU单元
	  CPU外频：CPU与外部组件进行数据传输/运算时的速度（玩“超频”的对象）
	  CPU倍频：CPU内部用来加速工作性能的一个倍速，两者相乘才是CPU的频率
	  “北桥总线”称为系统总线，因为是内存传输的主要信道，所以速度较快
	  “南桥”就是所谓的输入输出I/O总线，主要用于联系硬盘、USB、网卡等接口设备

======================

#3 -- 操作系统：包含“内核”和“系统调用”。即，操作系统其实就是内核与其提供的接口工具
	  
	  内核功能：
		1.系统调用接口 systme call interface
		2.程序管理 process control
		3.内存管理 Memory management
		4.文件系统管理 Filesystem management
		5.设备驱动 Device driver
